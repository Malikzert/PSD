Penyeimbangan dataset (SMOTE, ADASYN)#

Download data, simpan di MYSQL#

Gambar dibawah Repository** yang berisi data tentang protein localization sites. Pada halaman ini ditampilkan informasi ringkas seperti karakteristik dataset (multivariate), bidang studi (biology), jenis tugas (classification), jumlah instance (336 data), jumlah fitur (7 atribut), serta keterangan bahwa dataset tidak memiliki missing values. Untuk mendownload dataset ini, pengguna dapat menekan tombol DOWNLOAD (5.5 KB) yang berada di sisi kanan halaman, sehingga file dataset akan terunduh dalam format teks atau CSV yang bisa langsung digunakan untuk analisis maupun pemodelan machine learning.

Gambar di bawah adalah tampilan phpMyAdmin pada saat proses import dataset Ecoli (dalam format .csv) ke dalam MySQL. Setelah dataset diunduh dari UCI Repository, langkah selanjutnya adalah mengimpor file tersebut agar bisa disimpan dan dikelola di dalam database MySQL. Prosesnya sebagai berikut:

Masuk ke phpMyAdmin pilih database yang akan digunakan (misalnya ecoli_dataset).

Klik menu Impor.

Pada bagian Berkas untuk impor, klik Choose File lalu pilih file ecoli_dataset.csv yang sudah diunduh.

Pastikan Set karakter berkas diatur ke utf-8 agar data terbaca dengan benar.

Pada bagian Format, pilih CSV.

Scroll ke bawah dan klik tombol Go untuk memulai impor.

Setelah itu, MySQL akan membuat tabel berisi data dari file CSV tersebut. Jika file CSV belum memiliki struktur tabel (nama kolom), biasanya Anda perlu menyesuaikan pengaturan seperti Fields terminated by (misalnya , untuk koma) dan menentukan nama tabel atau kolom secara manual. Dengan cara ini dataset dari UCI dapat digunakan untuk query SQL, analisis, maupun integrasi ke aplikasi Anda.

Menghubungkan dan menginisialisasi dataset awal#

Kode Python di bawah bertujuan untuk mengunduh dataset Ecoli dari UCI Machine Learning Repository lalu menyimpannya dalam bentuk DataFrame menggunakan pustaka pandas. Bagian pertama memuat semua library yang dibutuhkan, termasuk pandas, numpy, matplotlib, seaborn, dan modul dari scikit-learn (StandardScaler, PCA), serta ADASYN dari imbalanced-learn untuk penyeimbangan data. Selain itu, ada juga sqlalchemy yang nantinya digunakan untuk koneksi ke database MySQL. Variabel konfigurasi untuk database seperti MYSQL_USER, MYSQL_PASS, MYSQL_HOST, MYSQL_PORT, dan MYSQL_DB sudah didefinisikan sehingga nantinya memudahkan proses penyimpanan data ke dalam MySQL.

Bagian berikutnya mendefinisikan URL dataset (ecoli.data) serta daftar nama kolom sesuai dokumentasi UCI, yaitu atribut-atribut seperti mcg, gvh, lip, chg, aac, alm1, alm2, serta class sebagai label kelas. Dataset kemudian dibaca menggunakan pd.read_csv() dengan parameter sep=r\s+ karena data dipisahkan oleh spasi/tab. Hasil pembacaan disimpan dalam DataFrame bernama df, lalu ditampilkan ukuran datanya dengan df.shape dan lima baris pertama dengan df.head(). Tahapan ini merupakan langkah awal untuk memastikan data berhasil diunduh dan terbaca dengan benar sebelum dilanjutkan ke tahap analisis lebih lanjut seperti PCA, visualisasi, maupun penyeimbangan data menggunakan ADASYN.

Visualisasi Data dengan PCA#

Kode di atas bertujuan untuk mengeksplorasi dan memvisualisasikan dataset Ecoli menggunakan dua pendekatan. Pertama, dibuat pairplot menggunakan seaborn untuk beberapa fitur numerik utama (mcg, gvh, lip, chg, aac, alm1, alm2) dengan label kelas sebagai hue. Pairplot ini menampilkan distribusi marginal tiap fitur di diagonal, serta sebaran dan korelasi antar fitur pada plot scatter di luar diagonal. Visualisasi ini membantu memahami pola awal, hubungan antar fitur, dan bagaimana kelas-kelas Ecoli tersebar di ruang fitur asli sebelum dilakukan transformasi dimensi.

Selanjutnya, data dinormalisasi menggunakan StandardScaler agar semua fitur memiliki skala yang sama, lalu dilakukan Principal Component Analysis (PCA) untuk mereduksi data menjadi dua dimensi (PC1 dan PC2). Hasil transformasi PCA disimpan dalam DataFrame baru df_plot, lalu divisualisasikan menggunakan scatter plot. Setiap titik mewakili satu sampel, dengan warna yang menunjukkan kelasnya. Plot PCA ini memudahkan identifikasi pola dan pemisahan antar kelas di ruang dua dimensi, sehingga bisa dibandingkan dengan sebaran asli dari pairplot, terutama untuk melihat overlap atau cluster dari masing-masing kelas.

import pandas as pd
import seaborn as sns
import matplotlib.pyplot as plt
from sklearn.preprocessing import StandardScaler
from sklearn.decomposition import PCA

features = ["mcg", "gvh", "lip", "chg", "aac", "alm1", "alm2"]
X = df[features]
y = df["class"]

df_pair = X.copy()
df_pair["class"] = y
sns.pairplot(df_pair, hue="class", corner=True, diag_kind="kde", palette="tab10")
plt.suptitle("Ecoli - Pairplot Fitur Asli", y=1.02)
plt.show()
scaler = StandardScaler()
X_scaled = scaler.fit_transform(X)

pca = PCA(n_components=2, random_state=42)
X_pca = pca.fit_transform(X_scaled)

df_plot = pd.DataFrame({
    "PC1": X_pca[:,0],
    "PC2": X_pca[:,1],
    "class": y
})

plt.figure(figsize=(8,6))
sns.scatterplot(data=df_plot, x="PC1", y="PC2", hue="class", palette="tab10", s=60, edgecolor="k")
plt.title("Ecoli - PCA (2D)")
plt.xlabel("PC1")
plt.ylabel("PC2")
plt.legend(bbox_to_anchor=(1.05,1), loc="upper left")
plt.tight_layout()
plt.show()

Penyeimbangan Data#

SMOTE#

Potongan kode ini digunakan untuk melakukan penyeimbangan kelas pada dataset Ecoli menggunakan metode SMOTE (Synthetic Minority Oversampling Technique). Pertama, dipilih fitur numerik utama sebagai X dan label kelas sebagai y. SMOTE kemudian diterapkan dengan parameter k_neighbors=1 untuk menghindari error pada kelas minoritas yang jumlah sampelnya sangat sedikit. Fungsi fit_resample dari SMOTE menghasilkan X_res dan y_res, yaitu dataset baru yang telah diimbangi sehingga setiap kelas memiliki jumlah sampel yang hampir sama. Kode juga menampilkan jumlah sampel per kelas sebelum dan sesudah balancing menggunakan pd.Series.value_counts() untuk memudahkan verifikasi efek oversampling.

Selanjutnya, kode menandai synthetic samples yang dibuat oleh SMOTE dengan membandingkan setiap baris hasil resampling X_res dengan data asli X. Fungsi rows_to_keys digunakan untuk mengubah setiap baris menjadi string unik berdasarkan nilai-nilai fitur yang dibulatkan, sehingga bisa dibandingkan secara tepat. Variabel is_synthetic berisi boolean yang menandai apakah baris tersebut merupakan sampel hasil generate oleh SMOTE (True) atau berasal dari data asli (False). Dengan ini, kita dapat mengetahui jumlah sampel baru yang ditambahkan, yang penting untuk visualisasi dan analisis lebih lanjut, misalnya ketika ingin menampilkan scatter plot PCA dengan menandai data synthetic.

from imblearn.over_sampling import SMOTE

features = ["mcg", "gvh", "lip", "chg", "aac", "alm1", "alm2"]
X = df[features].values
y = df["class"].values

smote = SMOTE(random_state=42, k_neighbors=1)
X_res, y_res = smote.fit_resample(X, y)

print("Before balancing:", pd.Series(y).value_counts().to_dict())
print("After  balancing:", pd.Series(y_res).value_counts().to_dict())

# tandai synthetic samples
def rows_to_keys(arr):
    arr_rounded = np.round(arr.astype(float), 8)
    keys = ["|".join(map(str, row)) for row in arr_rounded]
    return np.array(keys)

orig_keys = rows_to_keys(X)
res_keys  = rows_to_keys(X_res)
orig_key_set = set(orig_keys.tolist())
is_synthetic = np.array([k not in orig_key_set for k in res_keys])

print("Jumlah sampel resampled:", len(X_res))
print("Jumlah synthetic terdeteksi:", is_synthetic.sum())

ADASYN#

Potongan kode ini menunjukkan strategi gabungan untuk menyeimbangkan dataset Ecoli yang memiliki kelas ultra-minoritas. Pertama, digunakan SMOTE dengan k_neighbors=1 untuk menambah jumlah sampel di kelas-kelas yang sangat sedikit, sehingga setiap kelas memiliki minimal sampel yang cukup. Hasil resampling disimpan dalam X_sm dan y_sm, dan jumlah sampel per kelas ditampilkan menggunakan pd.Series.value_counts(), sehingga kita bisa melihat efek penambahan sampel sintetis dari SMOTE.

Setelah kelas ultra-minoritas cukup terwakili, diterapkan ADASYN pada dataset hasil SMOTE. ADASYN juga menggunakan n_neighbors=1 agar bisa berjalan tanpa error pada kelas minoritas kecil. Fungsi ini menambahkan sampel sintetis lebih adaptif berdasarkan distribusi kelas mayoritas di sekitar sampel minoritas. Hasil akhirnya disimpan di X_ada dan y_ada, dan jumlah sampel per kelas ditampilkan kembali. Dengan metode gabungan ini, dataset menjadi lebih seimbang dan siap untuk analisis atau visualisasi lebih lanjut, misalnya menggunakan PCA untuk melihat distribusi sampel asli dan sintetis.

from imblearn.over_sampling import SMOTE, ADASYN

# 1. SMOTE dulu untuk mengatasi kelas ultra-minoritas
smote = SMOTE(random_state=42, k_neighbors=1)
X_sm, y_sm = smote.fit_resample(X_scaled, y)

print("Setelah SMOTE:", pd.Series(y_sm).value_counts().to_dict())

# 2. Baru jalankan ADASYN di hasil SMOTE
adasyn = ADASYN(random_state=42, n_neighbors=1)
X_ada, y_ada = adasyn.fit_resample(X_sm, y_sm)

print("Setelah SMOTE+ADASYN:", pd.Series(y_ada).value_counts().to_dict())

Plot ini memudahkan identifikasi pola dan distribusi antara sampel asli dan hasil generate dari ADASYN. Kode di bawah ini digunakan untuk memvisualisasikan efek penyeimbangan kelas pada dataset Ecoli melalui tiga tahap: data asli, setelah SMOTE, dan setelah kombinasi SMOTE + ADASYN. Pertama, data fitur dinormalisasi menggunakan StandardScaler, kemudian dilakukan reduksi dimensi dengan PCA menjadi dua komponen utama (PC1 dan PC2). Fungsi get_synthetic_mask dibuat untuk menandai sampel sintetis yang dihasilkan oleh metode oversampling dengan membandingkan dataset hasil resampling dengan data asli. Data disimpan dalam DataFrame terpisah untuk tiap tahap, dengan kolom is_synthetic menandai sampel yang di-generate.

Visualisasi dibuat dalam satu figure dengan tiga subplot. Scatter plot pertama menampilkan distribusi data asli, diwarnai sesuai kelas. Plot kedua menunjukkan hasil SMOTE, di mana sampel asli tetap diberi warna kelas dan sampel sintetis ditandai dengan marker hitam x. Plot ketiga memperlihatkan hasil kombinasi SMOTE + ADASYN dengan cara yang sama. Dengan visualisasi ini, kita bisa dengan jelas melihat bagaimana sampel minoritas ditambahkan, distribusi kelas menjadi lebih seimbang, dan efek metode oversampling terhadap struktur data setelah reduksi PCA.

import matplotlib.pyplot as plt
import seaborn as sns
from sklearn.decomposition import PCA
from sklearn.preprocessing import StandardScaler
from imblearn.over_sampling import SMOTE, ADASYN
import numpy as np
import pandas as pd

features = ["mcg", "gvh", "lip", "chg", "aac", "alm1", "alm2"]
X = df[features].values
y = df["class"].values

scaler = StandardScaler()
X_scaled = scaler.fit_transform(X)

def get_synthetic_mask(X_orig, X_res):
    orig_keys = set(map(tuple, np.round(X_orig, 8)))
    res_keys  = list(map(tuple, np.round(X_res, 8)))
    return np.array([k not in orig_keys for k in res_keys])

pca = PCA(n_components=2, random_state=42)
X_pca = pca.fit_transform(X_scaled)

df_orig = pd.DataFrame({
    "PC1": X_pca[:,0],
    "PC2": X_pca[:,1],
    "class": y,
    "is_synthetic": False
})

smote = SMOTE(random_state=42, k_neighbors=1)
X_sm, y_sm = smote.fit_resample(X_scaled, y)

mask_smote = get_synthetic_mask(X_scaled, X_sm)

X_sm_pca = pca.fit_transform(X_sm)
df_sm = pd.DataFrame({
    "PC1": X_sm_pca[:,0],
    "PC2": X_sm_pca[:,1],
    "class": y_sm,
    "is_synthetic": mask_smote
})

adasyn = ADASYN(random_state=42, n_neighbors=1)
X_ada, y_ada = adasyn.fit_resample(X_sm, y_sm)

mask_ada = get_synthetic_mask(X_sm, X_ada)

X_ada_pca = pca.fit_transform(X_ada)
df_ada = pd.DataFrame({
    "PC1": X_ada_pca[:,0],
    "PC2": X_ada_pca[:,1],
    "class": y_ada,
    "is_synthetic": mask_ada
})

fig, axes = plt.subplots(1, 3, figsize=(20,6))

# Original
sns.scatterplot(data=df_orig, x="PC1", y="PC2", hue="class",
                s=40, alpha=0.8, edgecolor="k", ax=axes[0])
axes[0].set_title("Original Data")

# SMOTE
sns.scatterplot(data=df_sm[df_sm["is_synthetic"]==False],
                x="PC1", y="PC2", hue="class",
                s=40, alpha=0.8, edgecolor="k", ax=axes[1], legend=False)
synthetic = df_sm[df_sm["is_synthetic"]==True]
axes[1].scatter(synthetic["PC1"], synthetic["PC2"],
                c="black", marker="x", s=60, label="Synthetic")
axes[1].set_title("After SMOTE")
axes[1].legend()

# SMOTE + ADASYN
sns.scatterplot(data=df_ada[df_ada["is_synthetic"]==False],
                x="PC1", y="PC2", hue="class",
                s=40, alpha=0.8, edgecolor="k", ax=axes[2], legend=False)
synthetic = df_ada[df_ada["is_synthetic"]==True]
axes[2].scatter(synthetic["PC1"], synthetic["PC2"],
                c="black", marker="x", s=60, label="Synthetic")
axes[2].set_title("After SMOTE + ADASYN")
axes[2].legend()

plt.tight_layout()
plt.show()
